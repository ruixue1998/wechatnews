工信部 NVDB 提醒 Claude Code 安全后门隐患

According to the Network Security Threat and Vulnerability Information Sharing Platform of the Ministry of Industry and Information Technology (NVDB), the AI programming tool Claude Code has a security backdoor vulnerability, affecting versions 2.1.91 to 2.1.196. 据工业和信息化部网络安全威胁和漏洞信息共享平台（NVDB）消息，AI 编程工具 Claude Code 存在安全后门隐患，受影响版本为 2.1.91 至 2.1.196。

NVDB stated that Claude Code is an AI programming tool developed by Anthropic, which can complete code writing and repair based on text requirements; the affected versions have a built-in monitoring mechanism that can transmit sensitive information such as user region and identity to a remote server without user consent. NVDB 称，Claude Code 是 Anthropic 开发的 AI 编程工具，可根据文字需求完成代码编写和修复；受影响版本内置监控机制，未经用户同意即可向远程服务器回传用户地域、身份标识等敏感信息。

The platform recommends that relevant units and users conduct investigations, immediately uninstall or upgrade development terminals with affected versions to the latest secure version with relevant backdoor code removed, and strengthen external connection permission control and traffic monitoring for development tools within the core business network segment. 平台建议相关单位和用户开展排查，对安装受影响版本的开发终端立即卸载或升级至已清除相关后门代码的最新安全版本，并加强核心业务网段内开发工具外联权限管控与流量监测。

SpaceXAI 最强模型 Grok 4.5 发布，API 输出价格不到 Opus 4.8 的十分之一

SpaceXAI officially released Grok 4.5, positioned as the company's strongest model to date, focusing on coding, AI agent tasks, and knowledge-based work. The model was co-trained with Cursor, based on the new V9 foundation, with a parameter scale of approximately 1.5 trillion, and trained using tens of thousands of NVIDIA GB300 GPUs. SpaceXAI 正式发布 Grok 4.5，定位为公司迄今最强模型，主打编码、AI 智能体任务与知识型工作。模型与 Cursor 联合训练，基于全新 V9 基座，参数规模约 1.5 万亿，训练使用数万块英伟达 GB300 GPU。

In terms of performance, Grok 4.5 outperforms Claude Opus 4.8 on the Terminal-Bench 2.1 and DeepSWE 1.0 benchmarks, but is slightly lower on SWE-Bench Pro. Official data shows that for equivalent programming tasks, Grok 4.5's token consumption is about one-quarter of Opus 4.8's, and its service speed is 80 TPS. 性能方面，Grok 4.5 在 Terminal-Bench 2.1 和 DeepSWE 1.0 两项基准上优于 Claude Opus 4.8，但在 SWE-Bench Pro 上略低。官方数据显示，完成同等编程任务，Grok 4.5 的 token 消耗约为 Opus 4.8 的四分之一，服务速度为 80 TPS。

In terms of pricing, the API is priced at $2 for input and $6 for output per million tokens, with a context window of 500K tokens. The corresponding pricing for Opus 4.8 is $15 and $75, a significant difference. 价格方面，API 定价为输入 2 美元、输出 6 美元 / 百万 tokens，上下文窗口 500K tokens。Opus 4.8 的对应定价为 15 美元和 75 美元，差距显著。

法拉利营销总监：中国车企像是在做「快消品」，不是法拉利竞争对手

According to Malaysian automotive media DSF.my, Ferrari's Global Marketing Director Emmanuel Calando publicly stated yesterday that Chinese car manufacturers generally adopt a product model similar to Fast-Moving Consumer Goods (FMCG) – frequently launching new models, which quickly renders older cars in consumers' hands obsolete. 据马来西亚汽车媒体 DSF.my 报道，法拉利全球营销总监 Emmanuel Calando 昨日公开表态，称中国车企普遍采用类似快消品（FMCG）的产品模式——频繁推出新款车型，使消费者手中的旧车迅速过时。

He believes that this fundamentally differs from Ferrari's philosophy, which emphasizes long-term value and a unique driving experience. Therefore, Ferrari does not consider most Chinese high-end models as direct competitors. 他认为，这与法拉利强调长期价值和独特驾驶体验的理念存在本质差异，法拉利因此不将多数中国高端车型视为直接竞争对手。

Calando also pointed out that Chinese car manufacturers have made significant progress in straight-line performance, but there are still clear gaps in cornering precision, body posture stability, and emotional resonance in driving. Calando 同时指出，中国车企在直线性能上进步显著，但在弯道操控精准度、车身姿态稳定性以及驾驶情感共鸣方面仍有明显差距。

He stated that he frequently travels to China to test drive local models and has not yet found a product that can match Ferrari in terms of driving passion. He also admitted that the rapid iteration of Chinese car manufacturers is forcing Ferrari to continuously optimize its products. 他表示自己经常亲赴中国试驾本土车型，目前尚未发现能在驾驶激情上与法拉利相匹配的产品。他也承认，中国车企的快速迭代正在倒逼法拉利持续优化产品。

Data cited in the report shows that Ferrari delivered 584 vehicles in the Chinese market in 2025, a decrease of more than half compared to 1221 vehicles in 2023. Calando only took over this position last month; his predecessor's departure was directly linked to brand controversy triggered by the release of the first all-electric model, Luce, which led to a roughly 7% drop in the company's stock price. 报道援引的数据显示，法拉利 2025 年在中国市场交付 584 辆，较 2023 年的 1221 辆下降逾一半。Calando 于上月才接任这一职位，其前任离职的直接背景是首款纯电车型 Luce 发布后引发品牌争议，公司股价随之下跌约 7%。
